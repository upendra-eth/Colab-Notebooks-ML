#  Colab-Notebooks-ML
Welcome to my Machine Learning journey! This repository is a collection of my practice notebooks, experiments, and projects created using **Google Colab**.

###  My Goal
I am learning the fundamentals of Machine Learning, starting from data preprocessing to building and tuning predictive models.

###  Tools & Libraries
* **Language:** Python 
* **Platform:** Google Colab 
* **Libraries:** Pandas, NumPy, Matplotlib, Scikit-Learn

### 📂 Repository Structure
* `/Basics`: Introduction to Python for ML.
* `/Data_Preprocessing`: Cleaning and preparing data.
* `/Regression`: Linear and Logistic regression models.
* `/Classification`: Decision Trees, KNN, and SVM.
